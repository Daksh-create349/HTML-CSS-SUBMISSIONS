# Daksh Srivastava - Personal Portfolio

A personal portfolio website showcasing the skills and work of Daksh Srivastava.

## Features
- **Grid-Based Layout**: Organized sections for about me, skills, and projects.
- **Skillset Highlighting**: Lists proficiency in HTML, CSS, JavaScript, Python, Next.js, and more.
- **Interactive Elements**: Hover effects on "Big Cards" for a premium feel.
- **Clean Design**: Uses "Comic Sans MS" for a unique and friendly look.
