# Super Mario Welcomes 2026

A nostalgic Super Mario themed landing page designed as a portfolio or welcome scene for the year 2026.

## Features
- **Retro HUD**: Classic Mario-style heads-up display showing score, coins, and time.
- **Audio Control**: Interactive background music (Mario Theme) with ON/OFF switch.
- **CSS Animations**: Bullet Bill animation and dynamic text sequences.
- **Themed Design**: Uses "Press Start 2P" font and iconic pixel-art aesthetics.
