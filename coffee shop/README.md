# Coffee Culture Café

A warm and cozy website for "Coffee Culture", a place dedicated to true coffee lovers.

## Features
- **Themed Hero**: Engaging header with a vintage coffee aesthetic.
- **Full Menu**: Categorized list of Coffee, Tea, and Desserts.
- **Informative Sections**: About Us and Specialities showcasing the café's unique value.
- **Classic Design**: Comfortable and welcoming layout with a brown-toned palette.
