# SVG Shapes & Animations

A collection of Scalable Vector Graphics (SVG) experiments, showcasing custom shapes and SMIL animations.

## Features
- **Basic Shapes**: Lines, Rectangles, Triangles, and Curved paths.
- **Android Symbol**: A custom-drawn Android logo using SVG paths and circles.
- **Six-Pointer Star**: A complex star shape drawn with a single path.
- **Gradients**: Experiments with Linear and Path gradients.
- **SMIL Animations**: Moving circles, color transitions, and scaling effects.
