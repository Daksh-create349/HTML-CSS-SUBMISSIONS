# BOOKOphile — Where Readers Belong

BOOKOphile is more than just a bookstore. It's a curated digital space where stories inspire, writers grow, and readers connect beyond the pages.

## Features
- **Library**: Access to a curated collection of books.
- **Writers Hub**: A space for aspiring authors to share their work.
- **Community**: Reading circles and live events.
- **Modern UI**: Dark-themed, elegant interface with smooth hover effects.
