# Photography Channel

A visually appealing gallery website for photographers to showcase their work and "capture moments".

## Features
- **Interactive Gallery**: Grid layout with hover effects (overlay with "View Photo").
- **Hero Section**: High-quality imagery with a centered title.
- **Minimalist Navigation**: Sticky navbar for easy access to Home, Gallery, and Contact.
- **Responsive Layout**: Adapts seamlessly to different screen sizes.
