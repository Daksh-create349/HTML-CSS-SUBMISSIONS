# WaveRiders Surf Academy

A vibrant and energetic website for a surfing academy based by the ocean.

## Features
- **Course Catalog**: Highlights Beginner, Intermediate, and Pro surfing courses.
- **Dynamic Hero Section**: Bold imagery with a "Join Now" call to action.
- **About Section**: Explains the academy's mission and safety-first approach.
- **Image Gallery**: Showcases the surfing experience.
