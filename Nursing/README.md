# Compassionate Care Nursing

A professional and trustworthy website for a nursing services provider.

## Features
- **Services Grid**: Showcases In-Home Nursing, Elderly Care, and Post-Surgery Care.
- **Responsive Design**: Built to look great on all devices.
- **CTA Sections**: Easy navigation and clear calls to action for consultations.
- **Modern Aesthetic**: Uses a clean, medical-themed color palette.
